Projects

For this website I wanted a simple portfolio and blog administration engine that I could use to host a portfolio, and a basic blog engine. So I built my own using Flask and PostGreSQL. All of the code for this website can be found on my github and a full writeup is on the way.

A while back my church wanted to migrate their website from Weebly to Wordpress to reduce unnecessary operating costs. So I built a python tool to scrape the weebly blog archives, convert them to wordpress blog posts, and then submit them to the new website via the WordPress REST API. This one was a bit more difficult because there wasn't much documentation for the WordPress REST API, but thankfully I was able to reverse engineer it by studying the nework requests made by the browser when submitting new posts.

I got the idea to make a paper trading tool that would backtest different trading algorithms on previous stock price data, so I made this tool to do just that. It uses the yfinance module to download data from Yahoo Finance, and keeps the stock trading algorithms in seperate plug & play functions so that they can be swapped out at will. The code can be found on my github.
